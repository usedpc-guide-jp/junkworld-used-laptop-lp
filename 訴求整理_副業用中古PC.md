# 訴求整理：副業用中古ノートPC（Day2成果物）

作成日: 2026-04-14

---

## ターゲット像

- 副業をこれから始める、または始めたばかりの人
- 本業の収入以外に月数万円を作りたい
- PCは持っていないか、古くて重い/遅いものしかない
- 予算は3〜4万円（副業で回収できる範囲に抑えたい）
- ITリテラシーは普通（スマホは使えるがPC選びは詳しくない）

---

## 1. 何に使うのか

副業の作業環境として使う。具体的にはこの5つ。

| 用途 | 実際にやること |
|------|---------------|
| ブログ | WordPress管理画面での記事執筆・編集 |
| note | ブラウザ上での文章執筆・公開 |
| ChatGPT | ブラウザでの対話、プロンプト作成、文章生成の補助 |
| 画像作成 | Canva、ブラウザベースの画像編集（Photoshop級は不要） |
| 表計算 | Googleスプレッドシート、収支管理、簡易な分析 |

**共通点: すべてブラウザベースで完結する。**
ローカルに重いソフトをインストールする必要がない。

---

## 2. 何ができれば十分か（十分条件）

| 項目 | 十分ライン | 理由 |
|------|-----------|------|
| CPU | Core i5 第6世代以上 / Ryzen 3以上 | ブラウザタブ10枚+α が詰まらない |
| メモリ | 8GB | Chrome + Canva + スプレッドシート同時で不足しない |
| ストレージ | SSD 128GB以上 | HDDだと起動・操作すべてが遅くてストレス |
| 画面 | 13〜15.6インチ | 文章作業には最低13インチ、持ち運ぶなら14インチが限度 |
| バッテリー | 2時間以上持てば可 | 基本は電源接続で使う想定 |
| OS | Windows 10以上 | サポート・互換性の最低ライン |
| 重量 | 2kg以下 | カフェ持ち出しも視野に入る |

**一言: 「Core i5 / 8GB / SSD」が揃えば副業には十分。それ以上は贅沢。**

---

## 3. 何を避けるべきか（回避条件）

| 避けるべきもの | 理由 |
|---------------|------|
| メモリ4GB | ブラウザ3タブで限界。Canvaを開くと固まる |
| HDD搭載機 | 起動に2分、操作のたびに待ち時間。作業効率が半減する |
| Core i3 第4世代以下 | 処理が追いつかずブラウザがカクつく |
| 画面11.6インチ以下 | 文章作業で目と肩が疲れる。長時間の副業に向かない |
| バッテリー完全死亡 | 電源抜いた瞬間に落ちる。ACアダプタ必須になる |
| Windows 8以下 | セキュリティ更新が終了済み。ブラウザの対応も打ち切られていく |
| 保証なし・ジャンク品 | 初心者が検品できない。届いて動かないリスクが高い |

**一言: 「安いから」だけで選ぶと、メモリ4GB+HDDの地雷を引く。**

---

## 4. どこで失敗しやすいか（典型的な失敗パターン）

### 失敗1: 安さだけで選ぶ
→ 1万円台の激安に飛びつく → メモリ4GB+HDD → 遅すぎて使わなくなる → お金だけ消える

### 失敗2: スペック表を見ずに買う
→ 「Core i5」とだけ書いてあるが第3世代 → 10年以上前のCPU → 体感は遅い

### 失敗3: 保証なしの個人出品を買う
→ メルカリ・ヤフオクで「動作確認済み」を信じる → 1週間で不具合 → 返品不可

### 失敗4: 用途を決めずに買う
→ 「とりあえず安いやつ」で買う → 画面が小さすぎる/重すぎる/キーボードが打ちにくい → 結局使わない

### 失敗5: 高スペックを追いかける
→ 副業にはCore i7もメモリ16GBも要らない → 予算オーバーで始められない

**一言: 失敗の大半は「安さ優先」と「用途不明」の2つに集約される。**

## 5. 何を買えばよいか（推奨スペック帯）

### 推奨条件まとめ

```
価格帯:   3〜4万円
CPU:      Core i5 第6世代以上
メモリ:   8GB
SSD:      128GB以上
画面:     13〜15.6インチ
OS:       Windows 10以上
保証:     商品ごとに要確認（通常30日。一部は7日）
```

### なぜJUNKWORLDで探すのが合理的か

| ポイント | 内容 |
|---------|------|
| 価格帯 | 3万円台にCore i5/8GB/SSD搭載機の在庫が多い |
| 保証 | 通常30日保証。OS無しPC等は7日など例外あり |
| 検索性 | スペック条件で絞り込める |
| 在庫量 | 常時数千台規模で回転が早い |

### 探し方の手順

1. JUNKWORLDで「ノートPC」を選択
2. 価格帯を「3万円〜4万円」に絞る
3. メモリ「8GB以上」で絞る
4. SSD搭載を確認
5. CPU世代を型番で確認（Core i5-6xxxなら第6世代）
6. 画面サイズ・重量を確認
7. 在庫があるうちに決める（回転が早い）

## 訴求の優先順位（LP・広告で使う順番）

1. **用途の明確化** — 「副業用」と絞ることで自分ごと化させる
2. **十分条件の提示** — 「これで足りる」という安心を与える
3. **失敗パターンの回避** — 「これを避ければ失敗しない」で行動を後押し
4. **価格帯の妥当性** — 「3〜4万円で十分」は副業投資として合理的
5. **CTAは条件付き** — 「今の在庫で条件に合うものを見る」
